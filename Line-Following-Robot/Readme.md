#Line Following Robot
This Directory contains information, code and PCB design elements for ENEL602 Project 1 --- Line Following Robot

#Initial PCB creation
The initial PCB that I Subbmitted for part one of the project is found exported to a pdf in the file: "Altium Designer-P1_Josh."
For this part we were given a schematic and instructed to copy it and add relevent rules before moving onto the PCB layout. Once again for the PCB layout we were given the position for important component and then had to place the rest of the components and route traces between each component keeping within the project guidelines.

#Building the Robot
for the 2nd part of the project we had to assemble the robots.

#Coding the Robot
The code for from the microchip studio project is found in the folder "Robot Race"
For the 3rd and final part of the project we had to code the robot to follow a black line on a white backgroud. To get the best and most adabable pathfollowing robot i could i implimented a PID controller.
